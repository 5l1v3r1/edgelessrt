# OE_MutexDestroy()

Destroys a mutex.

## Syntax

    int OE_MutexDestroy(
        mutex);
## Description 

This function destroys a mutex that was initialized with OE_MutexInit().



## Parameters

#### Destroy

this mutex.

## Returns

Returns zero on success.
