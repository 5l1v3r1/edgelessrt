# OE_MutexLock()

Acquires a lock on a mutex.

## Syntax

    int OE_MutexLock(
        mutex);
## Description 

This function acquires a lock on a mutex.

For enclaves, OE_MutexLock() performs an OCALL to wait for the mutex to be signaled.



## Parameters

#### mutex

Acquire a lock on this mutex.

## Returns

Returns zero on success.
